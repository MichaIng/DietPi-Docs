# Release Notes

## March 2026 (version 10.2)

### Overview

The **March 2026** release of **DietPi v10.2** comes with **Immich** as a new software option, further enhancements and fixes.

### New software

- [**DietPi-Software**](../dietpi_tools/software_installation.md#dietpi-software) | [**Immich**](../software/cloud.md#immich) :octicons-arrow-right-16: This high performance self-hosted photo and video management solution has been added to our software catalogue. Available on x86_64 and ARMv8 only.

As always, many smaller code performance and stability improvements, visual and spelling fixes have been done, too much to list all of them here. Check out all code changes of this release on GitHub: <https://github.com/MichaIng/DietPi/pull/7989>
